# Transcribe Flow

# 📌 Description
Transcribe Flow is an AI-powered web application that converts audio into text and generates concise summaries using advanced speech recognition and NLP models.

# 🚀 Features
- 🎙 Audio Transcription – Convert audio files into text  
- 🧠 Automatic Summarization – Generate short summaries from transcripts  
- 📂 Multiple File Support – Upload audio files (WAV, MP3, etc.)  
- ⚡ Fast Processing – Quick transcription and summarization  
- 🌐 User-Friendly Interface – Simple and easy to use  
- 🔐 User Authentication – Secure login system  
- 📜 Download Results – Download transcription and summary  
- 🧩 AI Integration – Uses advanced AI models  


## 🛠 Technologies Used
- Python  
- Flask  
- HTML  
- CSS    
- OpenAI Whisper (Speech-to-Text)  
- Hugging Face Transformers (T5 Model)  
- MySQL  

# ▶️ How to Run
1. Clone the repository  
   https://github.com/ManuuB/TranscribeFlow  

2. Navigate to the project folder  
   cd TranscribeFlow  

3. Install dependencies  
   pip install -r requirements.txt  

4. Run the application  
   python app.py  

5. Open in browser  
   http://127.0.0.1:5000/
